关于系统

系统简介

MySQL 数据库课程实训 OJ 系统是一个基于 MySQL-WASM 8.0 的纯前端在线判题系统，旨在为 MySQL 数据库课程提供一个便捷的练习平台。

核心特点

  • 基于 MySQL-WASM 8.0，100% 兼容 MySQL 8.0 语法
  • 纯前端实现，无后端依赖，可直接部署到 Cloudflare Pages
  • 覆盖 80 道标准 MySQL 题目，按章节分类
  • 智能判题引擎，提供详细的错误提示和执行结果
  • 本地存储用户数据和做题记录
  • 响应式布局，支持 PC、手机、平板
  • 亮色/暗色模式切换

技术架构

纯前端 + MySQL-WASM 8.0（浏览器原生运行真实 MySQL 服务） + 本地 IndexedDB 持久化 + 静态文件部署

部署平台

Cloudflare Pages（静态托管，全球加速，永久免费）

使用说明

  1. 注册/登录系统
  2. 在题库页面选择题目
  3. 在题目详情页面编写和执行 SQL 语句
  4. 提交答案进行判题
  5. 在用户中心查看做题记录和统计信息
